# Graph Calculator 📈

A modern, high-performance, and visually stunning interactive graphing calculator. Designed with a premium "Glassmorphism 2.0" aesthetic, this tool allows you to visualize complex mathematical functions with ease.

## ✨ Key Features

- **Premium UI/UX:** Sleek obsidian theme with blurred panels and vibrant neon function curves featuring subtle glow effects.
- **Interactive Variable Sliders:** Use custom parameters like `a`, `b`, or `k` in your expressions. Sliders are automatically generated, allowing for real-time graph manipulation.
- **Auto-Animation:** "Play" your variables! Watch how parameters affect functions automatically with the built-in animation engine.
- **Collapsible Sidebar:** Maximize your graphing area with a dynamic sidebar that slides away when not needed.
- **Advanced Math Parser:** Supports implicit multiplication, common functions (`sin`, `cos`, `exp`, `log`, etc.), and custom variables.
- **Analytical Tools:**
    - **Table of Values:** Generate detailed data points for any function.
    - **Critical Points:** Automatically detect roots, intercepts, and local extrema.
- **Virtual Math Keyboard:** Built-in keypad for easy formula entry on any device.
- **Export as PNG:** Save your beautiful visualizations with a single click.

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to start exploring. No installation or build steps required!

### Basic Usage
1. Click **"+ New Expression"** in the sidebar.
2. Type a formula like `a * sin(b * x)`.
3. Adjust the sliders for `a` and `b` to see the graph update in real-time.
4. Click the **Play (▶)** button to animate the parameters.

## 🛠️ Technologies Used

- **HTML5 Canvas:** For high-performance, high-fidelity mathematical rendering.
- **Vanilla JavaScript:** Custom AST-based math parser and graphing engine (no heavy external libraries).
- **CSS3:** Advanced styling using modern CSS variables, backdrop filters, and grid layouts.
